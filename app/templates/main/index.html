В этом приложении можно посмотреть аналитику по non-life страховому рынку. Источник данных - файлы с сайта НБ РК.

Приложение опирается на четыре файла, загружаемых с сайта: Статистика > Показатели финансового сектора > Страховой сектор > Финансовые показатели

  • Сводные сведения о выполнении пруденциальных нормативов
  • Основные финансовые показатели страховых (перестраховочных) организаций Республики Казахстан
  • Поступление страховых премий по отраслям и классам страхования
  • Страховые выплаты по отраслям и классам страхования

На данный момент в приложение загружены данные за период с {{ moment(g.min_report_date).format('LL') }} по {{ moment(g.last_report_date).format('LL') }}

На вкладке Портрет компании можно посмотреть информацию о выбранной компании за выбранный период, а также сравнить основные показатели с рынком.

На вкладке Инфо по классу можно посмотреть информацию о выбранном классе страхования за выбранный период, а также динамику его развития.

На вкладке Обзор конкурентов можно сравнить показатели выбранной компании с основными конкурентами (за выбранный период). Конкурентов нужно будет выбрать из списка.

На вкладке Рэнкинг можно посмотреть позицию компаний по основным показателям.

На вкладке Автострахование можно посмотреть данные по автострахованию и страховому портфелю в целом.

Также вы можете оставлять посты, чтобы другие пользователи смогли увидеть ваши интересные находки и заключения. Посты будут видны всем пользователям.
